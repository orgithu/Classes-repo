# Ажлын хуваарь (4 хүний хуваарилалт)

## 1-р хүн (Өгөгдлийн сан ба эрхүүд)
- БД-ийн бүтэц боловсруулж, хүснэгтүүдийг байгуулсан: `customers`, `employees`, `packages`.
- Индексүүд болон анхны өгөгдлийн жишээг нэмсэн.
- `db.sql` дээр БД хэрэглэгчид болон эрхүүдийг (db_admin, db_web, db_customer_app, db_employee) тодорхойлсон.
- `db.sql` импортлохтой холбоотой асуудлуудыг засварлаж, схемийг нэг мөр болгон цэгцэлсэн.

## 2-р хүн (Нийтийн хайлт ба хэрэглэгчийн хэсэг)
- Нийтийн хайлттай `index.php`-г хийж, зөвхөн trackCode-аар хайх логиктой болгосон.
- `signup.php`, `login.php`, `dashboard.php` хэрэглэгчийн үндсэн урсгалыг хэрэгжүүлсэн.
- Хэрэглэгч зөвхөн өөрийн (утсаар холбогдсон) илгээмжээ харах логик нэмсэн.
- HTML формын validation болон `clean_input()` ашиглалт хийгдсэн.

## 3-р хүн (Ажилтны хэсэг)
- `employee.php` файлыг бий болгож, ажилтны багц нэмэх, хайх, picked тэмдэглэх, top-50 жагсаалт гаргах хэсгийг хийсэн.
- `includes/package.php`-д ажилтны үйлдлийн функцуудыг (add, search, mark_picked, list_top_packages) хэрэгжүүлсэн.
- Ажилтны нэвтрэлтийн логик болон session удирдлагыг цэгцэлсэн.

## 4-р хүн (Админ хэсэг + нийтлэг helper)
- `admin.php`-г DB админ зориулалттай болгож, хүснэгтүүдийг харах хэсгүүдийг нэмсэн.
- Админд зориулсан employee үүсгэх хэсэг, SQL console-ийн хэсгийг хэрэгжүүлсэн.
- `includes/customer.php`, `includes/employee.php`, `includes/auth.php`, `includes/db.php` туслах файлуудыг зохион байгуулсан.
- Системийн бүх файлууд хоорондын уялдаа холбоог шалгаж, багцласан.
